## ▶️ Getting Started

Clone the repository:

```bash
git clone <https://github.com/TofikTamboli/React-Learning--Journey->
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

or

```bash
npm start
```
